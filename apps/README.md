# Applications Directory — AgentForge

This workspace folder contains the deployable clients and gateway APIs of the AgentForge system.

---

## 📂 Sub-Workspaces

* **[Backend Gateway API (`apps/api/`)](file:///c:/Users/garvi/AgentForge/apps/api/):**
  A high-performance FastAPI service hosting the REST backend endpoints, WebSocket task execution streams, LangGraph multi-agent orchestration engines, and PostgreSQL connection pool migrations.
* **[Frontend Dashboard (`apps/web/`)](file:///c:/Users/garvi/AgentForge/apps/web/):**
  A Next.js 15 App Router web client offering visual progress tracking, team role model selectors, long-term memory explorers, and configuration pages.
* **[Console CLI Client (`apps/cli/`)](file:///c:/Users/garvi/AgentForge/apps/cli/):**
  A Python console client using the Click CLI library. Designed to let developers login, trigger ad-hoc file audits, and check task logs from terminal sessions.
